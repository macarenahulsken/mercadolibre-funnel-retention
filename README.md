## Business Problem
The company experiences user drop-off during the purchase journey and declining retention.

## Objective
Identify critical abandonment steps and evaluate user retention across countries.

## Analysis Performed
- Funnel conversion analysis
- Country segmentation
- Cohort retention (D7, D14, D21, D28)

## Key Findings
- 86% of users drop between product view and add-to-cart
- Paraguay shows 0% conversion after cart stage
- 76% of users churn within first 21 days

## Recommendations
- Optimize product page UX
- Improve onboarding experience
- Implement early engagement notifications

## Tools
SQL | Cohort Analysis | Product Analytics
